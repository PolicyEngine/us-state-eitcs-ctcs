# State EITC and CTC impacts

Interactive dashboard showing state-level Earned Income Tax Credit and Child Tax Credit impacts across US states.

## Architecture

- **Frontend**: React + TypeScript + Vite in `frontend/`
- **UI**: Inline CSS styles (no component library)
- **Maps**: MapLibre GL for interactive choropleth maps (state + district views)
- **Data**: Year-specific CSVs (`state_impacts_2025.csv`, etc.) + GeoJSON in `frontend/public/data/`
- **Deployment**: Vercel (static site)

## Development

```bash
cd frontend
npm install
npm run dev      # Dev server
npm run test     # Run tests
npm run build    # Production build
```

## Data pipeline

Year-specific CSV files are generated by the Modal pipeline using PolicyEngine US microsimulation. The CSVs contain state-level and congressional district-level impacts for CTCs, EITCs, and both combined.

### Modal Pipeline

The Modal pipeline runs simulations in parallel for faster execution:

```bash
cd scripts

# Login to Modal (first time only)
modal token new

# Run the full pipeline for a specific year
modal run modal_pipeline.py --year 2025

# Download results
modal volume get state-eitc-ctc-results state_impacts_2025.csv frontend/public/data/
modal volume get state-eitc-ctc-results district_impacts_2025.csv frontend/public/data/
```

### Pipeline Output

The pipeline generates year-specific files:
- `frontend/public/data/state_impacts_{year}.csv` - State-level impacts
- `frontend/public/data/district_impacts_{year}.csv` - Congressional district impacts
